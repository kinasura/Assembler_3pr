# Этап 4: Реализация арифметико-логического устройства (АЛУ)

## Цель
Завершить реализацию интерпретатора, добавив поддержку вычислительных операций.

## Выполненные задачи

### 1. Реализовано выполнение команды abs()
- Создан отдельный модуль АЛУ (`src/vm/alu.py`)
- Реализована операция `abs()` с поддержкой флагов состояния
- Добавлена обработка крайних случаев (переполнение)
- Интегрирован АЛУ с интерпретатором

### 2. Создана тестовая программа
- Разработана программа `examples/alu_test.asm`
- Демонстрирует вычисление абсолютного значения для различных случаев:
  - Положительные числа
  - Отрицательные числа  
  - Ноль
  - Крайние значения (минимальное/максимальное)
  - Различные смещения

### 3. Реализованы флаги состояния АЛУ
- **Z (Zero)**: результат равен нулю
- **N (Negative)**: результат отрицательный (для abs всегда false)
- **V (Overflow)**: произошло переполнение
- **C (Carry)**: перенос (пока не используется)
