== Формат команд
Команды представляются в CSV-формате с комментариями, начинающимися с ';'.

Пример:
----
158,679,28    ; Загрузить константу 679 в регистр 28
17,356,24     ; Прочитать из памяти по адресу 356 в регистр 24
----

== Команды

=== Загрузка константы (LOAD_CONST)
* Код операции: 158
* Формат: `158,<константа>,<адрес_регистра>`
* Размер в битах: 
  - A (0-7): 8 бит
  - B (8-37): 30 бит (константа)
  - C (38-42): 5 бит (адрес регистра)
* Пример: `158,679,28` → A=158, B=679, C=28

=== Чтение из памяти (READ_MEM)
* Код операции: 17
* Формат: `17,<адрес_памяти>,<адрес_регистра>`
* Размер: 5 байт
* Пример: `17,356,24` → A=17, B=356, C=24

=== Запись в память (WRITE_MEM)
* Код операции: 12
* Формат: `12,<регистр_адреса>,<регистр_значения>`
* Размер: 3 байта
* Пример: `12,5,3` → A=12, B=5, C=3

=== Абсолютное значение (ABS)
* Код операции: 214
* Формат: `214,<смещение>,<базовый_регистр>,<регистр_источник>`
* Размер: 5 байт
* Пример: `214,95,2,27` → A=214, B=95, C=2, D=27

== Ограничения
* Адреса регистров: 0-31 (5 бит)
* Константы: 0-1,073,741,823 (30 бит)
* Адреса памяти: 0-67,108,863 (26 бит для READ_MEM)
