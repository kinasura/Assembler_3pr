# Ассемблер и интерпретатор для учебной виртуальной машины (УВМ)

## Описание проекта
Реализация ассемблера и интерпретатора для учебной виртуальной машины согласно спецификации.

## Структура проекта
- `src/assembler/` - исходный код ассемблера
- `src/vm/` - исходный код виртуальной машины
- `tests/` - тесты
- `examples/` - примеры программ
- `docs/` - документация

## Установка и запуск
```bash
# Установка зависимостей
pip install -r requirements.txt

# Запуск ассемблера
python src/assembler/main.py examples/test.asm output.bin --test
